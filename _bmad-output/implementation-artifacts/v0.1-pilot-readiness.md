## Key Technical Fixes Landed During 49.5

1. **Turbo free-tier uploads** — `TurboFactory.unauthenticated()` is read-only (no `uploadFile`). Fix: ephemeral Arweave JWK via `arweave.crypto.generateJWK()` + `TurboFactory.authenticated({ privateKey: ephemeralJwk })` with zero-credit account. Turbo accepts ≤100KB uploads from zero-balance accounts without deposit. JWK is ephemeral — rotates on DVM restart, cannot be funded. [`docker/src/entrypoint-dvm.ts`]

2. **Connector self-route** — `routes: []` in connector.yaml means `getNextHop('g.townhouse')` returns null → F02 BEFORE `localDelivery` fires. Fix: patch connector.yaml to add `{ prefix: 'g.townhouse', nextHop: 'local', priority: 100 }`. [`packages/townhouse/src/__integration__/townhouse-dvm-arweave-e2e.test.ts`]

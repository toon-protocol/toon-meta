# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it
**privately** by emailing [dev.jonathan.green@gmail.com](mailto:dev.jonathan.green@gmail.com).

Do **not** file a public GitHub issue — public disclosure before a fix is in
place puts users at risk.

Include as much detail as you can: the affected component, a description of the
issue, and steps to reproduce if possible.

## Expected Response Time

You should receive an acknowledgement within **48 hours**. We will aim to
provide a fix or mitigation within **14 days** for high-severity issues,
depending on complexity.

Thank you for helping keep TOON Protocol secure.
